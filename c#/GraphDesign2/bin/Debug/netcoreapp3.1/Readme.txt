**you can only use write Int32 in textboxes,
otherwise program will be closed. (too high values cause errors)

**Program uses pixels to calculate and draw,
so you won't be able to use float values.

**It starts drawing from a default location
unfortunately user can't change :( 

**If you change background color, your paints
will be deleted.

**suggestion: Use it in fullscreen mode.

**NOTE:If you draw too big shapes some parts won't be visible.
